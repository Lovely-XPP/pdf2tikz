# pdf2tikz
Convert pdf (Vector Graph) to Tikz code.

## Requirments
Inkscape

## Usage
Set Absolute path of `inkscape.exe` (Windows) or `inkscape` (Linux or MacOS) to `pdf2tikz.py`.

For example:
```python
inkscape_path = "/Applications/Inkscape.app/Contents/MacOS/inkscape"
```
